# Exploratory Data Analysis Practice  

This repository contains multiple EDA projects to practice data analysis and visualization using Python.  

## Projects Included

- **Shark Tank India Season 1** – Analysis of investment trends, shark participation, deal structures, and equity patterns.  
- **Auto Sales Dataset** – Exploratory analysis of orders.  

## Tools & Libraries Used

- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Jupyter Notebook  

## Key Highlights

- Data cleaning, handling missing values, and type conversions  
- Visualization of trends and patterns  
- Correlation analysis and business insights  
- Episode-wise and investor-wise analysis (for Shark Tank India)
